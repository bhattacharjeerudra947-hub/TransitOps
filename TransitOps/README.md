# 🚍 TransitOps

TransitOps is a modern transport management web application designed to streamline transit operations through a clean, responsive, and user-friendly interface.

## ✨ Features

- User Authentication
- Role-Based Access
- Responsive UI
- Dashboard
- Modern React Interface

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- HTML5
- CSS3
- Git & GitHub

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/bhattacharjeerudra947-hub/TransitOps.git
```

Go to the project folder:

```bash
cd TransitOps
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open **http://localhost:5173** in your browser.
